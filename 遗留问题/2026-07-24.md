  - b. **前端改静态预构建**：本地 build 完 dist，直接 COPY 进镜像
  - c. **服务器升级到 4C4G**（~¥80/月）

- **可能的根因**（按概率排序）：
  1. nacos 服务没起来 / 健康检查没通过 / 端口不通
  2. application.yml 里 `spring.cloud.nacos.discovery.server-addr` 写的地址在容器里不通（容器名 vs IP vs localhost）
  3. datasource url 写的是 `localhost:3306`（容器里 localhost 是容器自己，不是宿主机）

- **需要现场抓**：
  - `docker logs <backend>` 完整输出
  - `docker exec backend ping nacos` / `ping mysql`
