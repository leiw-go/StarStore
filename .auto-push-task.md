## Deliverables

Write 4 PowerShell scripts + 1 .gitignore update + scheduled tasks. All scripts go in `E:\starstore\scripts\push-bot\`. Scripts SHOULD be tracked in git. Logs should NOT.

### 1. `watch-and-push.ps1` (real-time file watcher)

- Use `System.IO.FileSystemWatcher` on `E:\starstore`, include subdirectories
- Watch events: Created / Changed / Renamed / Deleted
- **Debounce**: 5 seconds of silence before triggering push (avoid save-spam)
- Push flow:
  - `git status --porcelain` — if empty, skip
  - `git add -A`
  - `git commit -m "auto: watcher @ <yyyy-MM-dd HH:mm:ss>"`
  - `git push origin master`
- Skip these paths to avoid loops:
  - `E:\starstore\.git\**`
  - `E:\starstore\scripts\push-bot\logs\**`
  - Any `*.log` files
- Error handling: catch git errors, write to log, do NOT crash the script
- Graceful shutdown on Ctrl+C
- Log file: `logs/watcher-YYYYMMDD.log` (rolling per day)

### 2. `hourly-push.ps1` (scheduled hourly push)

- One-shot script (run, do work, exit)
- Check `git log origin/master..HEAD --oneline` for unpushed commits
- If any: `git push origin master`
- If none: log "no changes" and exit 0
- Pull first? Yes, do `git pull --rebase --autostash origin master` before push, to avoid conflicts (only if there are local commits)
- Log file: `logs/hourly-YYYYMMDD.log`

### 3. `install.ps1` (register scheduled tasks)

Use `Register-ScheduledTask` to create TWO tasks:

**Task A: `StarStore-AutoPush-Watcher`**
- Trigger: AtLogon (user logon)
- Action: `powershell.exe -NoProfile -WindowStyle Hidden -ExecutionPolicy Bypass -File "E:\starstore\scripts\push-bot\watch-and-push.ps1"`
- Principal: current user, RunLevel Highest, LogonType S4U (or Interactive)
- Settings: AllowStartIfOnBatteries, DontStopIfGoingOnBatteries, RestartCount=3, RestartInterval=PT1M

**Task B: `StarStore-AutoPush-Hourly`**
- Trigger: Daily at 00:00, RepetitionInterval = PT1H, RepetitionDuration = Indefinite
- Action: `powershell.exe -NoProfile -ExecutionPolicy Bypass -File "E:\starstore\scripts\push-bot\hourly-push.ps1"`
- Same principal and settings as Task A

### 4. `uninstall.ps1` (remove scheduled tasks)

`Unregister-ScheduledTask` both tasks if they exist.
