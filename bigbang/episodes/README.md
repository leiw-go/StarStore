# 生活大爆炸 第一季 — 台词与常用短语整理

> 将 `source/` 下每一集的 `SHDBZ S01E**.srt` 中英双语字幕整理为按集存放的 Markdown 笔记。
> 解析脚本位于 `../parse_subs.py`。

## 目录结构

每个 Markdown 文件包含两大部分：

### 一、中英台词对照

- 按字幕序号依次列出每一条对白。
- 每条含：
  - 字幕时间戳（保留原始 `hh:mm:ss,ms` 时间点，便于回看视频片段）
  - **中文** 译文
  - **英文** 原文
- 已自动过滤掉片头片尾的广告水印条目（如「东西容器 / ABContainer / 微信公众号」）。

### 二、常用短语

| # | 英文 | 中文 | 出处 |
|---|------|------|------|
| … | 短语 | 译文 | 字幕条号 |

- 选取原则：
  - **长度**：不超过 6 个英文单词
  - **场景**：日常寒暄 / 礼貌用语 / 同意 / 疑问 / 回应 / 感叹等高频口语表达
  - **完整性**：必须以 `.` `!` `?` 收尾的完整语句，排除 `…, but` 之类的中途截断
  - **去重**：同义表达仅保留第一次出现的位置

## 使用建议

1. **按集学习**：以一集为单位，先读「常用短语」建立对话感觉，再回到「台词对照」精读上下文。
2. **时间戳定位**：每条对白都带时间戳，可以直接跳到视频对应位置反复听读。
3. **跨集累积**：同一短语在多集出现时，仅在第一次出现的集数中被收录——若想跨集汇总常用短语，可以基于每集的「常用短语」表再做一次 `group by 英文` 的合并。
