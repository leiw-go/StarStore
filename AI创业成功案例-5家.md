## 🏢 案例 2:DeepSeek(深度求索)

### 👤 创始人故事
- **梁文锋**:1985 年生于广东湛江,浙大信息与电子工程本硕
- 关键节点:
  - 2008 年:7 人团队用机器学习做量化交易,3 个月收益 500%
  - 2015 年:创立**幻方量化**
  - 2019 年:幻方管理规模破千亿(中国首家)
  - 2023.07:创立 DeepSeek,专注 AGI 研究
  - 2024.12:DeepSeek-V3 发布,引爆全球

### 👥 团队搭建
- 主体班底:**幻方量化系**(浙大背景 + 量化研究员)
- 强项:MLA 架构 + MoE 稀疏化(极致降本技术)
- 文化标签:**极客型 + 独立性强**(长期拒绝外部融资)
- ⚠️ 团队人数、具体岗位非公开

### 💼 商业模式
- **核心策略**:**开源 + 极低 API 价格**(API 价格仅为竞品 1/30)
- 不直接做 toC 产品,靠**开发者社区 + 学术界**扩散
- 资金来源:**幻方量化输血**(2025 年幻方平均收益 56.6%,当年为梁文锋带来 $7 亿收入)
- 路线:**先做技术影响力,再做商业化**

### 💰 融资结构
- **2026.07 首次启动融资**(之前一直拒绝资本):
  - 传闻规模:**500 亿元人民币(约 $73.5 亿)**
  - 估值:**不低于 $100 亿**
  - 梁文锋**个人出资 200 亿(40%)**
  - 腾讯拟出资 60 亿(2% 股权)
  - 国家集成电路产业投资基金(大基金)洽谈领投
- **意义**:从「独立理想主义」转向「资本协同」,DeepSeek V4 摆脱英伟达依赖

## 💡 关键洞察(总结)

### 1. 「学术派」与「工程派」两条路径
- **学术派**:唐杰(智谱)、闫俊杰(MiniMax)→ 偏研究驱动,周期长但护城河深
- **工程派**:梁文锋(DeepSeek)、Amodei 兄妹(Anthropic)→ 偏产品/技术工程化,迭代快

### 2. 「独立 vs 资本」的选择
- **长期独立**:DeepSeek(直到 2026.07 才启动融资)
- **快速拥抱资本**:Anthropic(5 年估值 $380B)、智谱(15-18 轮密集融资)
- **战略平衡**:MiniMax(阿里领投 + 港股上市)

### 3. 「产品 vs 技术」路线分化
- **产品驱动**:Manus(Agent 新品类)、MiniMax(Hailuo 视频)
- **技术驱动**:DeepSeek(MLA / MoE)、Anthropic(Constitutional AI)
- **平台驱动**:智谱(MaaS + GLM 全栈)

### 4. 共同的成功要素
- ✅ **创始团队具备深度技术积累**(非纯运营/资源型)
- ✅ **早期押注明确的技术信仰**(开源 / 安全 / Agent / 学术)
- ✅ **持续吸引顶级资本**(主权基金 / 顶级 VC / 战略巨头)
- ✅ **快速形成商业化闭环**(API / B 端 / C 端产品至少打通一条)
