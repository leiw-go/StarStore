# 网站 / APP / 小程序上线法务合规清单

> 适用场景：个人 / 中小团队 / 初创公司在国内面向公众发布互联网产品
> 适用法律：中华人民共和国（截至 2026 年 7 月）  
> 风险标签：🔴 必做（违法关停） | 🟠 高优（罚款/警告） | 🟡 中优（合规建议） | 🟢 建议

---

## 〇、三个"必须先想清楚"的事

1. **先备案后上线**：网站/APP 备案号下来前不要对外公开
2. **先定主体再开发**：建议注册有限责任公司，避免个人无限连带责任
3. **先看类目再开发**：医疗 / 金融 / 新闻 / 教育 / 视听等类目有强制前置审批，不是改一行代码能解决的

## 一、共性合规基线（三形态通用）

### 1. 🔴 经营资质
| 资质 | 适用条件 | 主管部门 | 备注 |
|------|----------|----------|------|
| **ICP 备案** | 任何接入公网的域名 | 工信部 | 先备案后接入，未备案会被运营商屏蔽 |
| **公安联网备案** | 网站上线 30 日内 | 属地公安网安 | 免费，可在 [beian.mps.gov.cn](https://beian.mps.gov.cn) 申请 |
| **ICP 经营许可证（EDI）** | 经营性业务（付费、广告、会员、电商） | 通信管理局 | 注册资金 ≥ 100 万（部分省份 1000 万），外资受限 |
| **APP 备案** | APP 上架前 | 工信部 | 2024 年起强制，与 ICP 备案不同 |

### 2. 🔴 个人信息保护（《个保法》）
- 必须有**《隐私政策》**和**《用户协议》**；APP 内 4 步内可触达
- **最小必要**：能邮箱注册就别要求手机+实名+人脸
- 每项权限（位置、通讯录、相麦、相机等）需**运行时单独申请 + 单独同意**
- 敏感个人信息（人脸、身份证、医疗、行踪、未成年人）需**单独同意**
- 收集个人信息的种类、用途、共享清单、用户权利、联系方式需完整披露
- 提供**查询 / 更正 / 删除 / 导出 / 注销**路径（通常 15 个工作日内响应）
- 涉及 ≥ 100 万人个人信息的，需做**PIIA（个人信息保护影响评估）**
- ⚠️ **罚款上限**：5000 万元 或 上年营业额 5%

### 3. 🟠 数据安全与跨境（《数据安全法》《网络数据安全管理条例》）
- 数据分类分级（重要数据须识别，报主管部门）
- 数据出境三条路径：① 安全评估（重要数据/100万人/100万人敏感）② 标准合同 ③ 个人信息保护认证
- **境外服务器不能直接装 GDPR/CCPA 像素**：Google Analytics、Facebook SDK 在国内可能触发审查
- 等保 2.0：金融/医疗/政府/能源等行业强制，一般 C 级系统可选
- 数据加密、备份、访问日志留存（建议 ≥ 6 个月）
- ⚠️ 发生数据泄露需**立即报告**：向网信部门和受影响用户

### 4. 🟠 内容合规（《网络安全法》《互联网信息服务管理办法》）
- 禁止 9 类违法不良信息（涉政、涉黄、涉暴、谣言、虚假宣传等）
- UGC（用户生成内容）需建立**「通知-删除」避风港**机制
- 跟帖评论、弹幕、论坛、群组类需**先审后发**（网信办 2022 年新规）
- 涉及公众账号 / 直播 / 弹幕 / 网络表演等需持相应许可
- 推荐建立：关键词过滤 + 人工审核 + 黑名单库 + 投诉入口

### 5. 🟠 知识产权
- **商标**：核心业务名 + Logo 提前在**国家知识产权局商标局**注册（35 类广告销售、42 类科技服务、9 类 APP、38 类通信、41 类文娱、45 类社交按需）
- **域名**：注册 .com / .cn / .com.cn 全套防抢注
- **软件著作权**：APP 源码做完即登记（拿软著），下证 1-3 个月
- **版权风险重灾区**：
  - 字体（方正、汉仪、华康、苹方商用均需授权；思源黑体免费）
  - 图片（Unsplash、Pexels、StockSnap 等 CC0 协议可用）
  - 音乐（商业音乐平台或 CC0 音乐库）
  - 地图（高德/百度/腾讯地图需申请 API key）
- 用户上传内容侵权：建立**"避风港"机制**——接到通知即删除即可免责

### 6. 🟠 广告合规（《广告法》《互联网广告管理办法》）
- 所有广告内容必须标"**广告**"或"推广"（包括信息流、软文、贴片、视频框中框）
- 禁止**虚假宣传**、**绝对化用语**（"最佳""第一""唯一""国家级"）
- 禁用"幸存者偏差"、"权威推荐"等诱导话术
- **重灾区**：功效宣称、治疗作用、招商加盟、刷单炒信
- 特殊行业广告需**前置审批**：
  - 医疗、药品、医疗器械 → 卫健委
  - 保健食品 → 市场监管总局
  - 金融（理财、贷款、保险）→ 银保监 + 金融监管局
  - 教育培训 → 教育部门（义务教育阶段禁止）
  - 房地产 → 必须标注预售证号

### 7. 🟡 算法与 AI
- **算法备案**：具有舆论属性或社会动员能力的算法（推荐、排序、检索、生成、调度）→ [beian.cac.gov.cn](https://beian.cac.gov.cn)
- **生成式 AI 服务**（《生成式人工智能服务管理暂行办法》）：必须**算法备案 + 安全评估**，训练语料合规
- **深度合成内容**：需显著标识"由 AI 生成"
- 不得用 AI 自动生成新闻、不得操纵算法干预舆论

### 8. 🟡 主体与税务
- **主体选择**：个体户（轻但无限责任）vs 有限责任公司（推荐）vs 个人独资（税务成本可能更低但也是无限责任）
- **对公账户 + 税务登记**是硬需求，私户收款 = 偷税
- 经营性收入需建账、开发票（季度报税）
- 雇佣员工需要：劳动合同、社保、个税代扣
- 海外用户 / 美元收入 → 需考虑外汇合规（5 万美元额度）

## 二、三种形态差异化要求

### 🌐 网站
| 项 | 要点 |
|----|------|
| 备案 | ICP 备案 + 公安联网备案 |
| 经营许可 | 经营性 → EDI（ICP 经营许可证） |
| Cookie 弹窗 | 必需，且需与隐私政策联动 |
| 域名 | 备案需指向中国大陆境内服务器或备案接入商 |
| ICP 黑洞 | 境外服务器域名需通过 CDN 备案接入（阿里云/腾讯云/华为云提供） |
| 跳转 | 跨境引流需提前备案，否则被举报后接入商断网 |
| 内容 | 评论/UGC 需装网安过滤系统（如网易易盾、阿里云反垃圾） |

### 📱 APP
| 项 | 要点 |
|----|------|
| 备案 | ICP 备案 + 公安备案 + **APP 备案**（工信部 2024 新规） |
| 隐私政策入口 | 首启动页 + "我的"页面 + 设置页内可触达（≤ 4 步） |
| 权限申请 | 必须**运行时申请** + 弹窗说明 + 单独同意 |
| 第三方 SDK | 全部必须清单化披露（SDK 名、用途、收集字段、第三方主体、官网） |
| 最低 SDK 版本 | 强制支持最新权限 API（Google 13+ 隐私沙箱、Apple 隐私标签） |
| 上架审核 | 应用商店（华为/小米/OPPO/vivo/Apple Store）需提交：<br>① 隐私政策 URL ② 权限使用说明 ③ 应用截图 ④ 著作权证书（部分商店） ⑤ ICP 截图 |
| 数据收集 | 关闭 `IDFA / GAID` 时不得绕过收集；Apple 的 App Tracking Transparency 必弹 |
| 更新 | 每次发版需做合规自检（应用商店扫描 + 内部 SOP） |
| 海外 | Google Play / App Store 需满足 GDPR / CCPA / COPPA 等地区合规 |

### 💬 小程序
| 项 | 要点 |
|----|------|
| 备案 | 服务器域名仍需 ICP 备案 + 公安备案（依托平台，不代表豁免） |
| 类目资质 | 微信类目分级（信息查询、工具、生活服务、医疗、教育、金融等），部分类目需提交资质证明（营业执照、行业许可证、ICP 备案截图等） |
| 用户隐私 | 必须配置《小程序用户隐私保护指引》+ 小程序弹窗 |
| 支付 | 用微信支付自动合规；第三方支付要走开通类目 |
| 登录 | 强制用户授权弹窗，授权后必须提供**「解除授权」入口** |
| 跳转外部 | 跳 H5 / 下载 APP 需申请对应类目（"外部链接"原则：腾讯备案制） |
| 更新 | 每次发版需平台审核（微信 1-7 天） |
| 注销账号 | 必须提供注销路径，且数据彻底删除 |

## 三、上线前准备清单（按优先级执行）

### 🔴 第一步：主体与基建（第 1-4 周）
1. ☐ 注册**有限责任公司**（拿到营业执照 + 5 个章）
2. ☐ 注册域名、做 **ICP 备案**（7-20 个工作日）
3. ☐ 申请对公账户、税务报到（小规模 vs 一般纳税人选型）
4. ☐ 注册**自有商标**（35 / 42 / 9 / 38 类按需，审查 6-9 个月）
5. ☐ 服务器/云服务合同（境内大厂：阿里云、腾讯云、华为云）
6. ☐ 软件著作权登记（下证 1-3 个月，建议早提交）

### 🟠 第二步：必备法律文件
7. ☐ **《用户服务协议》**：权利义务、行为守则、免责声明、争议管辖（推荐约定被告所在地法院）
8. ☐ **《隐私政策》**：收集清单、用途、共享清单、用户权利、联系方式（DPO 或合规岗位）
9. ☐ 《儿童信息保护规则》（如涉及 14 岁以下）
10. ☐ 《Cookie 政策》（如使用）
11. ☐ 《平台社区规则》（如有 UGC）
12. ☐ 主播实名协议 / 创作者协议 / 商家入驻协议（按业务）

### 🟡 第三步：内容与运营
13. ☐ 制定《内容审核规范》（含违规图库、关键词库）
14. ☐ 搭建**关键词过滤 + 人工审核 + 投诉下架**机制
15. ☐ 实名认证 + 反作弊（短信 / 人脸 / 活体，按需）
16. ☐ 用户投诉 / 申诉受理渠道（一般在 7 日内响应）

### 🟢 第四步：技术与安全
17. ☐ 等保测评（如适用，金融/医疗强制，其他建议二级起步）
18. ☐ 数据加密（HTTPS、TDE、AES-256）+ 备份 + 访问日志
19. ☐ 个人信息保护影响评估报告（PIIA）
20. ☐ 应急响应预案（数据泄露、网安事件、舆情处置）
21. ☐ 准备 SDK 合规清单（用于商店审核与隐私政策披露）

### 🔵 第五步：上架
22. ☐ 应用商店上架准备（隐私政策 URL、权限说明、截图、著作权证书）
23. ☐ 小程序类目资质提交（经营类目对应的前置许可证）
24. ☐ 公安联网备案
25. ☐ APP 工信部备案

|------|----------|----------|------|
| **新闻信息服务** | 互联网新闻信息服务许可证 | 国家网信办 | ⭐⭐⭐⭐⭐（基本批不下来给小公司） |
| **网络出版** | 网络出版服务许可证 | 国家新闻出版署 | ⭐⭐⭐⭐⭐ |
| **信息网络传播视听节目** | 信息网络传播视听节目许可证 | 国家广电总局 | ⭐⭐⭐⭐⭐ |
| **网络文化经营** | 网络文化经营许可证 | 文化部 | ⭐⭐⭐（直播 / 表演类） |
| **互联网医疗** | 互联网医疗信息服务 + 诊疗（双证） | 卫健委 | ⭐⭐⭐⭐（建议靠医院合作） |
| **药品交易** | 互联网药品交易服务资格证（向个人） | 药监总局 | ⭐⭐⭐⭐⭐ |
| **金融理财** | 银行 / 支付 / 小贷 / 保险 / 基金牌照 | 金融监管总局 | ⭐⭐⭐⭐⭐（建议先 SaaS 化） |
| **金融服务** | 备案或牌照（按类型） | 金融监管总局 | ⭐⭐⭐⭐ |
| **药品 / 医疗器械广告** | 药品广告审查表 | 市场监管总局 | ⭐⭐⭐ |
| **教育培训** | 办学许可证（义务教育阶段另议） | 教育部门 | ⭐⭐⭐ |
| **电信增值业务** | EDI / ISP / VPN 等 | 通信管理局 | ⭐⭐⭐ |
| **人力资源** | 人力资源服务许可证 | 人社部 | ⭐⭐ |
| **拍卖** | 拍卖经营批准证书或备案 | 商务局 | ⭐⭐⭐ |
| **广告** | 广告发布登记（发布端） | 市场监管局 | ⭐ |

> 💡 **冷启动建议**：如果行业涉及前置许可，先用 "工具 / 助手" 形态低门槛切入，不要直接做交易 / 内容 / 资质服务平台。

## 五、罚款与责任速查表

| 违规情形 | 处罚依据 | 罚款 / 后果 |
|----------|----------|-------------|
| 未备案经营 | 《电信条例》 | 罚款 1-5 万，情节严重 10 万+ |
| 违规收集个人信息 | 《个保法》第 66 条 | 100 万 / 50 亿用户；5000 万 / 5% 营业额 |
| 数据泄露 | 《数据安全法》 | 100 万-1000 万；吊销许可 |
| 未做等保 | 《网络安全法》第 21 条 | 5 万-50 万 |
| 滥用算发推荐 | 《算法推荐管理规定》 | 1 万-10 万（连续累计可至 100 万） |
| 未成年保护不力 | 《未成年人保护法》 | 5 万-50 万；吊销许可 |
| 虚假广告 | 《广告法》 | 广告费 3-5 倍罚款；情节严重吊销 |
| 生成式 AI 未备案 | 《暂行办法》 | 罚款 + 关停 |
| 拒不接受网信办检查 | 《网络安全法》第 69 条 | 5 万-50 万；法律责任人另罚 |

## 六、易踩坑点 Top 10

1. **域名备案了但服务器实际在境外** → 一举报就断网
2. **隐私政策链接打不开 / 404 / 过期** → 商店审核打回
3. **超范围收集权限**（e.g. 工具 APP 调通讯录） → 商店打回 + 监管罚款
4. **SDK 用了但隐私政策没披露** → Apple 审核打回 + 国内举报
5. **第三方字体直接商用** → 律师函 + 索赔（方正起诉过大量 APP）
6. **用户协议没约定管辖法院** → 异地法院诉讼，律师费翻倍
7. **删除条款不清** → 行政调证时还要配合；用户因数据留存投诉
8. **不接受"通知删除"机制** → 作为平台被卷进案件，承担侵权连带责任
9. **退出登录 ≠ 注销账号** → 用户注销无路径，违反必要原则
10. **海外用户服务范围含欧盟** → 没 GDPR 适配就是 4% 营业额罚款

## 八、落地节奏建议（建议马上做的 5 件事）

按下来一周内可全部完成：

1. **明晚 30 分钟**：决定主体形态（个人 / 个体户 / 有限公司）+ 注册一家公司
