# 程序员 → 外贸公司：正向现金流盈利路线图

> 目标人群：程序员背景、零外贸经验、希望从兼职试水到全职稳定的过渡
> 核心目标：**90 天内跑通链路 + 第一笔正向现金流**，不是"做一个品牌"
> 关键约束：不囤货、不雇人、不烧大钱，直到月利润稳定 > 3 万

---

## 2. 程序员的隐藏优势（别浪费）

你以为你的优势是"写代码"。其实你的优势是下面这些**和外贸直接相关的**：

- **选品调研**：爬 Amazon、1688、TikTok 数据，做对比分析（你已经在 `E:\CrawlCode\` 干过这事）
- **独立站技术栈**：Shopify Hydrogen / Next.js 自建站，省 1-3 万外包费
- **客服自动化**：ChatGPT + 邮件模板 + 半自动工单系统，一个人顶 3 个客服
- **数据看板**：Google Sheets + Apps Script，省掉一个 BI 工具的钱（年省 5000+）
- **广告投放**：Facebook Ads / Google Ads API 自动化，比手动调省 30% 成本
- **物流/支付 API**：Stripe、PayPal、连连、PingPong 都能写脚本对接

**关键认知：你的代码能力不是用来"做产品"，是用来"省人力成本"。** 一个程序员外贸人，可以做到 5 万月利润的规模而不需要雇全职员工——这是非技术背景的外贸人做不到的。

**行动项：** 列一个表，列出你**已经会用**的工具/技能（vs 不会的），后面建站、自动化、广告时优先用熟悉的。

## 3. 三种入门模式（按现金流速度排序）

下面三种模式，**不要一开始就锁定**。先看完三种，再决定。

### 模式 A：服务型外贸 / 代运营（最快出单）

**你做什么：** 帮中国工厂/品牌处理海外渠道——建站、Listing、客服、社媒运营、广告投放。

| 维度 | 数值 |
|------|------|
| 启动资金 | < 1 万 |
| 第一笔收入时间 | 1-3 个月 |
| 现金流周期 | T+0 ~ T+30 |
| 月利润天花板（1 人） | 1-3 万 |
| 风险 | 客户流失、被跳单 |

**适合：** 想 1-3 个月看到第一笔收入、不想囤货、想先熟悉外贸链路的人。
**最大坑：** 客户做大了甩掉你，所以**一定要收预付款 + 月费**，而不是纯佣金。

### 模式 B：轻资产一件代发（Dropshipping / Print on Demand）

**你做什么：** 独立站 + 广告投放 + 供应商一件代发（客户下单 → 供应商直发）。

| 维度 | 数值 |
|------|------|
| 启动资金 | 1-3 万（主要用于广告测试） |
| 第一笔收入时间 | 1-3 个月 |
| 现金流周期 | T+7 ~ T+30 |
| 月利润天花板（1 人） | 1-5 万（爆款运气成分大） |
| 风险 | 广告费 > 售价、供应商断货、品控失控 |

**适合：** 想做"自有品牌 + 渠道"但没钱囤货的人。
**最大坑：** 不要上来就 All-in 单一爆款（你不知道哪个会爆），要**小批量测款**（每个 SKU 投 50-100 美金广告测试）。

### 模式 C：自有产品 + 海外仓（重资产）

**你做什么：** 选品 → 国内采购 → 头程运输 → 海外仓 → 多渠道销售。

| 维度 | 数值 |
|------|------|
| 启动资金 | 20 万+ |
| 第一笔收入时间 | 3-6 个月 |
| 现金流周期 | T+30 ~ T+90（库存占款） |
| 月利润天花板 | 5-50 万 |
| 风险 | 库存滞销、汇率波动、平台规则变更 |

**适合：** 资金到位、想长期做品牌的人。
**最大坑：** 90% 的新人死在这一步——不是产品不行，是**库存占款吃光了现金流**。

### 我的推荐组合：**模式 A 起步 + 模式 B 试探**

- 模式 A 让你**3 个月内就有现金流**（哪怕月入 5000 也是正向）
- 模式 B 用业余时间**测试自有渠道**，6 个月后看数据决定要不要转模式 C
- **不要上来就做模式 C**——除非你已经验证过模式 A 或 B 的链路跑得通

**行动项：** 选定一个模式（推荐先选 A 或 B），写在 `personal/盘点.md` 顶部。

## 4. 市场初筛（基于"程序员 + 现金流优先"两个约束）

不要看"哪个市场最大"——要看**"哪个市场最适合你这种背景"**。

| 市场 | 平台 | 难度 | 客单价 | 现金流速度 | 合规复杂度 | 推荐度 |
|------|------|------|--------|------------|------------|--------|
| 🇺🇸 美国 | Amazon FBA、Shopify | ⭐⭐⭐ | 高 | 中 | 中 | ⭐⭐⭐⭐ |
| 🇸🇬 东南亚 | Shopee、Lazada、TikTok Shop | ⭐⭐ | 中低 | 快 | 低 | ⭐⭐⭐⭐⭐ |
| 🇦🇪 中东 | Noon、Amazon SA | ⭐⭐⭐ | 很高 | 中 | 中 | ⭐⭐⭐ |
| 🇲🇽 拉美 | Mercado Libre | ⭐⭐⭐ | 中 | 中 | 高（语言） | ⭐⭐ |
| 🇪🇺 欧洲 | Amazon EU、CDISCOUNT | ⭐⭐⭐⭐ | 高 | 慢 | 高（VAT） | ⭐⭐ |

**推荐：先选 1 个国家 + 1 个平台**，不要同时铺。

- **兼职试水首选：** 东南亚（Shopee/Lazada）—— 中文友好、规则简单、起量快
- **有一定资金首选：** 美国（Amazon FBA）—— 客单高、规则成熟、数据工具全
- **追求高客单价：** 中东（沙特/阿联酋）—— 但要会阿拉伯语或用翻译工具

**行动项：** 选定 1 个国家 + 1 个平台，不要超过 1 个。写在盘点文档里。

- [ ] **建站**（Shopify 起步，1-2 周搞定；不要上来就自研）
- [ ] **合规清单**（进出口权备案、品牌商标、平台资质）

### Phase 3：MVP 上线（W7-W10）

- [ ] 上架 5-20 个 SKU（不要超过 20 个，先测款）

### Phase 4：稳定 + 优化（W11-W12）

- [ ] 把客服、订单处理自动化（写脚本 / 用工具）

### Phase 5：规模化判断（> W12）

- [ ] **月利润 > 1 万**：可以考虑雇兼职/外包部分工作
- [ ] **月利润 > 3 万**：可以考虑小批量囤货 / 海外仓

- [ ] **月利润 < 5000**：不要扩张，复盘链路问题

## 6. 现金流管理（你最关心的一节）

外贸新人 90% 不是死在"没生意"，是死在"现金流断裂"。下面 6 条是底线：

### 6.1 区分"利润"和"现金流"

- **库存占款 ≠ 利润**：100 万的货，可能只赚 20 万，但 100 万压在那里
- **应收账款 ≠ 现金**：客户付了款但你没收到前，都不算
- **平台账期**：Amazon 14 天结算一次，Shopee 7-15 天，不要忽略

### 6.2 现金流红线

- **家庭刚性支出 × 12** = 你的现金储备底线。低于这个数就停止扩张。
- **月利润 vs 月现金流**：第一个月看现金流（实际到账的钱），不要看账面利润。

### 6.3 第一年目标阶梯

| 阶段 | 月净利润目标 | 对应投入 |
|------|--------------|----------|
| 0-3 个月 | > 0（跑通） | 1-3 万试水盘 |
| 3-6 个月 | > 5000 | 加测品、加渠道 |
| 6-12 个月 | > 1-3 万 | 考虑小批量囤货 |
| 12 个月+ | > 3-5 万 | 雇人/海外仓/品牌 |

### 6.4 不要做的事

- ❌ **不要囤货**（除非月利润已经稳定 > 3 万 6 个月以上）
- ❌ **不要租办公室**（家里办公足够）
- ❌ **不要雇全职员工**（先外包，按单结算）
- ❌ **不要做品牌**（先做渠道/铺货，验证链路再说）
- ❌ **不要做平台**（先做独立站或单平台深耕）
- ❌ **不要忽略外汇结算和税务**（必找代理记账）

**行动项：** 用 Google Sheets 建一个**现金流表**，每周更新：库存占款、应收账款、应付账款、家庭支出。

## 8. 常见陷阱（提前知道能救命）

| 陷阱 | 后果 | 怎么避 |
|------|------|--------|
| 一上来就做"品牌" | 烧光钱 | 先做渠道/铺货 |
| 同时铺 5 个平台 | 哪个都没跑通 | 只做 1 个平台 |
| 囤货赌爆款 | 库存压垮 | 只测不囤，验证后小批量 |
| 选品靠"我看好这个" | 没数据支撑 | 必须用工具数据选品 |
| 第一个月就想"对标 XXX 大卖" | 心气崩 | 第一个月目标是"跑通链路" |
| 忽略平台规则 | 封号 | 每月读一次平台政策更新 |
| 一个人干所有事 | 累垮 | 财务对账、客服等可外包 |
| 信用卡、广告费全靠垫付 | 现金流断 | 现金储备要够 6 个月 |
